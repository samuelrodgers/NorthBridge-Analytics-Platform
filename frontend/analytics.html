Q5 · Running company revenue over time

Performance query — window function
Q5 — f_transaction → d_time → d_company · cumulative window
Buckets revenue by minute, computes cumulative sum per company using a window function (SUM OVER PARTITION BY c_id ORDER BY minute_bucket). This drives the live dashboard refresh graph. Most sensitive to index availability on d_time — expect the largest before/after delta in performance benchmarks here.
